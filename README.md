# Cyber-101

This repository is a growing collection of my notes on cybersecurity fundamentals. ######

###  The Core Goal: The CIA Triad
Every security strategy aims to protect three main things:
* **Confidentiality:** Keeping secrets safe so only authorized people can view them (e.g., encryption, passwords).
* **Integrity:** Making sure data is accurate and has not been tampered with by unauthorized users.
* **Availability:** Ensuring systems and data are up and running when users need them.

###  Common Types of Cyber Threats
To defend a system, you must know what you are defending against. Here are the most common attacks:
* **Malware:** Malicious software like viruses, ransomware, and spyware designed to harm or lock systems.
* **Phishing:** Social engineering attacks using fake emails or websites to trick people into giving up sensitive information.
* **DDoS (Distributed Denial of Service):** Flooding a server with massive amounts of fake traffic to crash it.
* **Man-in-the-Middle (MitM):** An attacker secretly intercepting and reading communication between two parties on a network.
* **Injection Attacks:** Forcing a web application to execute malicious code, such as an SQL Injection against a database.

###  Basic Prevention & Defense Mechanisms
Security is built in layers. Here are the most common ways to protect digital assets:
* **Strong Authentication:** Using complex passwords alongside Multi-Factor Authentication (MFA).
* **Network Defenses:** Using firewalls to block bad traffic and VPNs to secure remote connections.
* **Cryptography:** Encrypting sensitive data so it remains unreadable even if it gets stolen.
* **Patch Management:** Consistently updating software and operating systems to fix known vulnerabilities.
* **Principle of Least Privilege:** Giving users and programs only the exact minimum access they need to do their job.

###  The Approach: Red Team vs. Blue Team
The cybersecurity industry generally splits the approach into two main perspectives:
* **Red Team (Offensive Security):** Ethical hackers who actively try to break into systems to discover weak spots before the bad guys do (Penetration Testing).
* **Blue Team (Defensive Security):** Security analysts who build defenses, monitor networks for suspicious activity, and respond to live attacks.
